# Ethermint Faucet Frontend

Frontend that works with this [Faucet](https://github.com/hanchon/faucet).

## Configuration

Set the oauth env variables:
`REACT_APP_AUTH0_DOMAIN`
`REACT_APP_AUTH0_CLIENT_ID`

## Build

```sh
yarn install
yarn build
```

## Run locally

```sh
yarn install
yarn start
```
